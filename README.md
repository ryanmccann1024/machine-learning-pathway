  - Statistics
    - [The Elements of Statistical Learning](https://hastie.su.domains/Papers/ESLII.pdf)

    - [The Matrix Cookbook](http://www2.imm.dtu.dk/pubdb/edoc/imm3274.pdf)

      - Linear regression
      - Logistic regression
      - K-nearest-neighbors
      - Support vector machines
